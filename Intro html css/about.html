PROGRAM PSEUDOCODE

START
Input house number
Input Amount Paid
PAYMENT VERIFICATION PROCESS
IF Payment = Paid AND Yes Debt
Display Reject notice
ELSE IF Payment = Paid AND no Debt
THEN DATE PROCESS
IF Date = Early
Set Discount = amount*0.1
Display Acknowledgement notice
ELSE IF Date = Late
Set Penalty = amount*0.2
Display Penalty Fee
ELSE IF Date = Due
THEN WATER DISTRIBUTION PROCESS
IF Rent Payment = Full Payment
Set Water = 24/7
Display List
ELSE IF Rent Payment = Half Payment
Set Water = Morning and Evening hours only
Display List
ELSE IF Rent Payment = No Payment
Set Water = Morning hours only
Display List
END
